## 🔭 Overview

Welcome to the **GIS Research Repository**! This project is a curated, searchable collection of 40 critical GIS research topics, ranging from urban heat island mapping and climate change analysis to advanced machine learning for spatial predictions.

Originally a single-file implementation, this platform has been evolved into a high-performance **Next.js 15 App Router** application. It serves as a bridge between theoretical GIS concepts and practical implementation, providing researchers, students, and urban planners with the exact steps, tools, and datasets needed to execute complex spatial workflows.

### 🌟 Key Features

- **40 Deep-Dive Topics**: Categorized across Environment, Urban Planning, Disaster Management, Agriculture, and more.
- **Implementation Blueprints**: Step-by-step guides for QGIS and SAGA GIS workflows.
- **Dataset Catalog**: Direct links to 80+ global datasets (Landsat, Sentinel, WorldClim, etc.).
- **Plugin Registry**: Detailed installation and usage instructions for essential QGIS plugins (SCP, QNEAT3, etc.).
- **Common Errors & Fixes**: A built-in troubleshooting guide for every topic to help you overcome common GIS pitfalls.
- **Interactive UI**: A performance-optimized, dark-themed interface with fluid animations and real-time filtering.

---

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a custom futuristic dark theme.
- **Animations**: [Framer Motion](https://www.framer.com/motion/) for staggered grid entrances and layout transitions.
- **Components**: Built on [Radix UI](https://www.radix-ui.com/) primitives for maximum accessibility and reliability.
- **Type Safety**: 100% [TypeScript](https://www.typescriptlang.org/) for robust data management.

## 🚀 Getting Started

To run this repository locally:

1. **Clone the repo**
   ```bash
   git clone https://github.com/JustJhong609/GIS-Research.git
   cd GIS-Research
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🤝 Contributing

We are looking for GIS enthusiasts, spatial analysts, and developers to help grow this repository! Whether you want to add new topics, update dataset links, or improve the implementation steps, your contributions are welcome.

### How to Contribute:
1. **Fork** the repository.
2. **Create a branch** for your feature (`git checkout -b feature/new-gis-topic`).
3. **Commit your changes** (`git commit -m 'Add Topic: Hydrological Connectivity'`).
4. **Push to the branch** (`git push origin feature/new-gis-topic`).
5. **Open a Pull Request**.

### Contribution Areas:
- **Data Enrichment**: Adding more specialized plugins or common error solutions to `data/topics.ts`.
- **New Topics**: Expanding the repository beyond the initial 40 topics.
- **UI Enhancements**: Improving the accessibility or visual polish of the application.
